SCSS = Sassy Cascading Style Sheet
- Sassy : 엄청난, 놀라운, 엽기적인

1.기존 스타일시트가 가지고 있지 못한 기능 구현
- CSS > Nesting 도입
- extends

2.프로그래밍 언어 친화적
- 함수 구현 => 반복적인 무언가를 신속 & 효율적으로 처리

3.Compiler 언어
- interpreter 언어 : JS => 개발자 어떤 코드 => 즉각적으로 읽어서 구현 실행 // 굉장히 실행 // 로딩속도 => 만약 개발자가 잘못된 코드를 작성 => 즉각 실행 => 오류 // 손실 발생!!!

- compiler 언어 : Java, C => 해석 후 반영  => 로딩속도 x // 만약 개발자의 실수 // 컴파일 자체 x // 실행 x // 오류 x => 안전성!!!

- 선생님 뭐가 안됩니다!!! => 90% 오타!!!! // 문법 x

*SCSS는 기본적인 CSS 문법을 따릅니다!!! + @

1. Nesting 기능
2. Ampersand 응용기능_1 (Ampersand = &)
3. Ampersand 응용기능_2 (@at-root)
- 부모로 부터의 상속을 일시적으로 벗어날 수 있도록 해주는 기능
4. Style Group 생성 및 활용 (@mixin // @include)
5. @mixin 응용 > 매개변수 (arguments => 동전 코인)
- 공통된 스타일 속성을 각각의 요소에 독립적으로 적용해주는 형태로 컴파일링 됩니다.
- 선택자가 중심 => 스타일을 불러모으는 효과

6. @extends > 확장하다
- 공통된 스타일 속성안으로 각각의 요소들을 불러서 사용!!!
- 스타일이 중심 => 선택자를 불러모으는 효과
