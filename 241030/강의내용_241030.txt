- static : 정적인 <-> 동적인
> html 태그를 입력한 순서에 맞춰서 순서대로 좌표를 유지하는 것!!
> html에서 default(*기본값)

- relative : 상대적인 <-> 절대적인
> 원래 static값을 가지고 있는 현 상태를 기준으로 상대적으로 어떤 좌표값인지를 결정

- absolute : 절대적인
> 현재 사용자가 보고있는 웹 브라우저 화면을 기준으로 절대적인 좌표값 부여

> 예외사항!!!
만약, 절대적인 위치값을 주고 싶긴하나, 특정 요소를 기준으로 한 상태에서 절대값을 적용하고 싶은 경우!!!

- fixed : 고정된
> 다른 요소들이 스크롤바에 의해서 위치가 이동될 때, 유일하게 고정된 좌표값을 적용받는 요소!

- sticky : 부착되어있는 (*sticker)
> 특정 위치에 도달하기 전까지는 기존 static 속성을 그대로 유지하다가 해당 특정 위치에 도달하면 그때에는 스티커처럼 부착되는 형태를 유지!!

- overflow : hidden
> 특정 구역을 정의해놓고 해당 구역 바깥에 있는 요소는 보이지 않게 하고자 할 때
> overflow : scroll

- z-index : 
> 만약 position 속성값으로 absolute(*절대적인)  좌표값이 여러개 존재할 때, 어떤 요소를 최우선으로 나타나게 하고, 후순위로 어떤 요소를 나타나게 하고자 할 때
